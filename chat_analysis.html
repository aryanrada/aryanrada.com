Whatsapp Chat Analysis

About Project

The web-based application is used to analyze the chats of users individually and in groups. this application provides further analysis that users can easily understand using the graphs used for visualization.

This application is developed in python with the use of different libraries which include streamlit, matplotlib, seaborn, urlextract, wordcloud, pandas, and emoji.

User and Role description

User:

The user uploads their chat data and the system does the rest. It will clean and organize user data and visualize using different graphs for better understanding.

System Features

  • Data Gathering
  • Data Cleaning
  • Data Organizing
  • Data Visualization
  • Use of Algorithms
